# 写真選定計画

ステータス: 実装前選定フェーズ

photo-selectorは使用しません。掲載写真は、人間が目視で選定し、掲載許可・スマホトリミング・表示速度を確認してから実装します。

## 選定の最優先基準

LPに使う写真は、Instagramで反応が良い写真ではなく、予約・相談につながる写真を優先します。

判断順:

1. 見た人が「自分たちもこう残してもらえそう」と感じるか
2. ASAKA Photographyの思想が伝わるか
3. スマホで主役が伝わるか
4. 相談前の不安が減るか
5. 文字やCTAの邪魔をしないか
6. 掲載許可が明確か

## 1. Hero写真選定

### 役割

Hero写真は、サービス説明より前に、依頼者の未来の記憶へ接続する写真にします。

見た瞬間に伝えたいこと:

- 何年後かに見返したくなる
- その日の幸せまで思い出せそう
- 写真が綺麗なだけでなく、時間が残っている
- ASAKA Photographyに任せたら、自然に過ごせそう

### 最優先候補の条件

- 人物同士の関係性が見える
- 表情、距離感、手元、後ろ姿のいずれかに感情がある
- スマホ縦長で成立する
- 主役が遠すぎない
- 文字を載せられる余白がある
- 暗すぎず、肌や表情が沈まない
- ロケーションが主役になりすぎない

### 避ける写真

- 景色だけが強い写真
- 人物が小さすぎて感情が伝わらない写真
- 文字を載せる余白がない写真
- 「綺麗な場所で撮った写真」に見えるだけの写真

### 現在確認できた素材への判断

実LP資料側で確認できた `susuki-sunset-desktop.webp` / `susuki-sunset-mobile.webp` は、夕景や空気感の候補としては使えます。

ただし、人物の関係性や「今日の幸せまで思い出せる」感情接続が弱い場合、Heroの第一候補ではなく、Signatureや余韻の写真として再評価します。

## 2. About写真選定

### 役割

About写真は、朝香大樹という人への信頼を作る写真にします。

見せたいこと:

- 撮る前に人を見ている
- 距離感を急に詰めない
- 必要なときだけ整えている
- 撮影中の空気が穏やか
- 依頼者が安心して過ごしている

### 候補写真

優先:

- 朝香大樹が撮影中に声をかけている写真
- 被写体と適切な距離で向き合っている写真
- 子どもや家族を急かさず見守っている写真
- 手元や立ち位置をやさしく整えている写真

補助:

- 自然なプロフィール写真
- カメラを構えている横顔や後ろ姿
- 撮影現場の空気が分かる引き写真

避ける:

- 自己紹介感が強すぎる写真
- カメラマンが主役になりすぎる写真
- 腕組み、強い目線、硬いポーズのプロフィール写真

## 3. Works写真選定

### 役割

Worksは作例一覧ではなく、人生の流れとして見せます。

表示順:

1. Wedding
2. Maternity
3. Newborn
4. お宮参り
5. 七五三
6. Family

### 各ジャンルの基準

#### Wedding

- ふたりの関係性が見える
- キメカットだけでなく、会話や距離感が残っている
- ロケーションより、ふたりの時間が主役になっている

#### Maternity

- 体調への配慮を感じる
- お腹だけでなく、これから家族になる空気が伝わる
- パートナーや上の子との関係性があると強い

#### Newborn

- 赤ちゃん単体だけでなく、家族が見守る空気がある
- 無理な演出に見えない
- 安心、静けさ、小ささが伝わる

#### お宮参り

- 赤ちゃん、親、祖父母など家族の関係性が見える
- 儀式感と家族の温度が両立している
- 神社や場所の情報より、家族の節目が主役になっている

#### 七五三

- 子どもの背伸び、照れ、甘えが見える
- 整った記念写真だけでなく、その年齢らしい表情がある
- 親子の距離感が伝わる

#### Family

- 子どものペースを活かしている
- 抱っこ、走る、甘える、遊ぶなど「家族の今」が見える
- 家族全員の視線や手元に関係性がある

## 実装前チェック

各写真ごとに確認すること:

- 掲載許可
- LP掲載範囲
- 顔出し可否
- スマホ縦トリミング
- PC横表示
- ファイルサイズ
- altテキスト
- 類似写真の並びすぎ
- 色味の統一

## 推奨ファイル配置

```text
src/assets/photos/
  hero/
  about/
  works/
    wedding/
    maternity/
    newborn/
    omiyamairi/
    shichigosan/
    family/
  signature/
  cta/
```

Astroの画像最適化を使う写真は `src/assets/photos/` に置きます。  
URL固定が必要なOGPやロゴは `public/images/` に置きます。
